# B+树
## definition
一个order为M的B+树：
1. 根节点是叶节点或者有2-M个孩子
2. 非叶节点（除了根节点）有ceil(M/2)-M个孩子
3. 所有叶节点有相同的depth
4. 叶节点上键值的个数为ceil(M/2)-M

所有叶节点的键值按从小到大顺序排列
非叶节点上存储的是除了第一颗子树的最小键值

## insert
插入到叶节点，需要的话不断向上分裂
最先插入的M个节点看作一个叶子，当插入第M+1个节点时，分裂成两个节点，每个节点ceil(M/2)个键值

## delete
删除叶节点中相应的数据，如果：
a. 不满足ceil(M/2)个键值，则从兄弟节点借一个
b. 如果兄弟节点也不满足，则和兄弟节点合并
